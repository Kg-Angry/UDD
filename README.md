# UDD_UPP

**Uvod:**

Predmetni projekat podrazumevao je razvoj samostalne web aplikacije koja simulira rad Naucne centrale unutar koje se rukuje dokumentima (casopisi i radovi) iz pripadajucih naucnih oblasti - pretraga dokumenata vrsi se uz oslonac na ElasticSearch. Postupak objavljivanja naucnog rada u casopisu modelovan je pomocu poslovnog procesa koji je realizovan primenom Camunda alata. 

**Radno okruzenje:**

* Visual Studio Code
* Intelij IDEA

**Tehnologije:**

* Angular 6
* Spring Boot 2.2.0 Release
* ElasticSearch
* Camunda
