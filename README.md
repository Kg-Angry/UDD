# UDD

**Uvod:**

Predmetni projekat podrazumevao je razvoj samostalne web aplikacije koja simulira rad Naucne centrale unutar koje se rukuje dokumentima (casopisi i radovi) iz pripadajucih naucnih oblasti - pretraga dokumenata vrsi se uz oslonac na ElasticSearch.

**Radno okruzenje:**

* Visual Studio Code
* Intelij IDEA

**Tehnologije:**

* Angular 6
* Spring Boot 2.1.0 Release
* ElasticSearch v5.6.16 sa SerbianAnalyzer-om
